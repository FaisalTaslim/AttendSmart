| What is AttendSmart?

A Universal Smart Attendance Tracking System is a web-based attendance monitoring solution designed for multiple sectors, including schools, colleges, and companies.The system automates attendance tracking using multiple methods like location-based tracking, wifi-ssid tracking, and QR-code scan to maintain efficiency.

Note: Read carefully!

AttendSmart is currently optimized for desktops and smartphones. While some smaller tablets may work, we recommend avoiding tablet use for the best experience. Dashboard

AttendSmart provides users with three dashboard based on the major roles they hold inthe organization

Admin Dashboard

  1. It helps to manage AttendSmart on an organizational level.
  2. Admin can manage users â€” add, remove, or archive users.
  3. Admin can manage roles â€” create, delete, and add roles to users. Roles make it easier to search users based on their job description, and even manage them.
  4. It allows the admin to view leave requests of users, and accept or reject the requests.
  5. It allows the admin to view attendance records, graphs, and download them in a .pdf format.

Employee Dashboard

  1. It helps to manage AttendSmart on an employee level.
  2. It allows the employees like teachers to view attendance records, graphs, and download them in a .pdf format.
  3. All staff can mark their own attendance from this dashboard.
  4. Teachers can accept or reject a student's leave requests.

Student Dashboard

  1. Students can mark their attendance through the Dashboard.
  2. They can view their attendance statistics, and even download it in a .pdf format.
